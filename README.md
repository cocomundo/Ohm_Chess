# Chess Engine - Ohm Chess

This is an chess engine based on minimax and a/b-pruning, represented by a curses chessboard.
a/b-pruning and performance analysis are switchable 

### Dependencys:

ncurses


### To build this project:
```
$ cd Ohm_Chess
$ make
```
